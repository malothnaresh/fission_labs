Steps followed are:

  • First, checked out the angular-seed app from npm to kick start the project, instead of defining the folder structure and others from start point
  • Now, added lodash, highcharts, bootstrap, jqeury to support bootstrap
  • Provided a input field for reading the csv file
  • After file uploaded, loading its contents from a directive named fileReader
  • Once file contents are rendered.. in function readCsv, parsing, validating and invoking drawChart is handled
  • Alternatives tried for csv to json conversion : csv2json package, jquery-csv package
  • But, they are not much developer friendly. So, designed my own parsing technique
